Graph Database Built Right

Dgraph is an open source, distributed, and highly available graph database, built to scale horizontally.

Try itTake a tour

  • Low Latency, High Throughput

    Low enough latency to serve real time queries over terabytes of structured data.

  • Distributed and Scalable

    Easily scale to multiple machines thanks to the sharded storage and minimal network calls.

  • Open Source

    Written entirely in Go, Dgraph is available under open-source license.

Why Dgraph?

Dgraph is a graph database built to scale horizontally. Its sharded storage and query processing were specifically designed to minimize the number of network calls.
